# 你好！这里是个人的《逗比的人生》的低技术力开源延续版！
## 这是什么？
- 这是一个基于python语言和json数据搭建的简易纯文本的人生模拟游戏
- 你将扮演一个普通的逗比，在无数随机的事件中做出选择，在无穷可能的特质中寻找组合，完成稀有事件或是获取稀有特质
- 是根据记忆和情怀，《逗比的人生》非官方的精神延续，完全开源版本，避免任何商业应用
- 旨在不仅尽量完整地还原游戏内容，同时为所有感兴趣的玩家提供一个平台参与进来，可以一起添加或完善事件，让本游戏在共同的创作下愈发完整

## 快速开始
### 方式一
直接运行可执行文件。  
对于Windows用户，请双击main目录下的`main.exe`  
对于Linux（Ubuntu测试通过），请运行main目录下的`main`
```bash
cd main
./main
```
> 可执行文件相对main文件更新不定期，若有版本较低问题，建议您选择方式二或是联系我
### 方式二
#### 前置条件 
- Python 3.10 或更高
- 安装依赖：  
>1. pip install blessed
>2. pip install json5
>3. pip install pyyaml


#### 启动游戏
```bash
# 克隆仓库（也可以手动下载）
git clone https://github.com/Flame-PD/doubilife.git
cd doubilife
# 安装依赖
pip install blessed
pip install json5
pip install pyyaml
#运行
python main.py
```
或使用项目中的脚本
- `Linux/macOS/WSL`: `./start.sh`
- `Windows`: `start.bat`
> 目前仅在`WSL`上测试过，不保证可在其他平台运行，如出现兼容性问题请联系我
### 进入游戏后  
请使用键盘，通过输入指令的方式游玩。主菜单界面输入2可以退出游戏，或者可以直接关闭终端窗口。  
建议终端宽度在`25 * 100`及以上，本地测试环境为`30 * 110`，仅供参考。

## 核心实现
### 事件(Event)
事件是游戏内主要经历的部分，一个事件包含标题、描述、选项列表以及触发条件。玩家将不断地经历事件，并做出对应选择
一个事件可以有子事件，但子事件不可以加入事件池
### 选项(Option)
选项是玩家可选择的行动。每个选项可以包含：
- 显示文本（包括选项描述和选择后的结果）
- 触发条件（需要哪些特质/不能有哪些特质）
- 分支结果，指示产生的效果
### 特质(Trait)
特质是玩家拥有的状态特色，直接影响事件的产生和选项的效果，大部分条件都包含是否拥有某个特质。而特质本身也是一种“成就徽章”，少见的特质也是收集的乐趣之一
### 分支(Branch) 与 结果(Result)
依赖于选项存在的结构，一个选项会调用结果，而结果会遍历分支，找到符合条件的一个
### 事件调度
使用队列和池共同维护，队列保存着“即将到来的事件”，维护子事件
池保存着所有当前合法的、可以被取出的事件，只有队列为空时才会从池中不放回地随机抽取，随机为权重制

## 项目结构
```text
doubilife/
├── core/                      # 核心代码
│   ├── event.py               # 事件、选项、分支定义
│   ├── player.py              # 玩家状态管理
│   ├── game.py                # 游戏逻辑实现
│   ├── condition.py           # 实现所有条件判断
│   ├── loader.py              # 实现从data内读入json
│   └── event_container.py     # 事件调度（库/池/队列）
├── data/                      # 游戏数据部分
│   ├── package/               # 目前计划按照包为单位导入事件和特质
│   │   ├── standard/          # 至少包含的基础包
│   │   └── example/           # 一个示例包
│   ├── config.yml             # 决定导入哪些包
│   └── README.md              # 创作者请看  
├── main.py                    # 入口
├── README.md                  # 你正在看的部分
└── LICENSE                    # GPLv3
```
